# Bhaii❤️ - Personal Health Assistant

A warm and friendly AI-powered personal health buddy built with Python and Groq API.

## Features
- Casual, friendly health advice
- Asks clarifying questions before responding to serious concerns
- Remembers conversation context
- Never diagnoses or prescribes medication

## Setup
1. Clone this repo
2. Install dependencies:
   pip install groq python-dotenv
3. Create a .env file and add your Groq API key:
   GROQ_API_KEY=your-key-here
4. Run the chatbot:
   python Bhaii_chatbot.py
